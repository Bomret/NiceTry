Exchange<T>(Try<IEnumerable<T>>)

Returns all values in the specified <em>tryEnumerable</em> as
<xref href="NiceTry.Try`1" data-throw-if-not-resolved="false"></xref>, if it contains an enumerable. If
<em>tryEnumerable</em> represents failure, an enumerable containing only
that failure is returned.
